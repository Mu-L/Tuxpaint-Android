Rita med Tux "Magic" Verktyg: Sned välj

Grupp Projektioner

Upphovsperson: Bill Kendrick <bill@newbreedsoftware.com>

Klicka (och dra) för att positionera den vinkel som ska användas med verktyget "Oblique Draw".

Detta verktyg används på fri hand.

Detta verktyg är endast tillgängligt när "komplexitet" (expertnivå) är inställd på "Avancerad".

Se även: 1-punktsritning, 1-punktsval, 2-punktsritning, 2-punktsval, 3-punktsritning, 3-punktsval, Isometrisk linje, Dimetriskt val, Dimetrisk ritning, Trimetrisk välj, Trimetrisk ritning, Snedritning, & Perspektiv.
