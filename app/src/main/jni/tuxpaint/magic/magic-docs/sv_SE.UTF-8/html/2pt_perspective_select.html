Rita med Tux "Magic" Verktyg: 2-punktsval

Grupp Projektioner

Upphovsperson: Bill Kendrick <bill@newbreedsoftware.com>

Klicka (och dra) för att placera de två försvinnande punkterna som ska användas med verktyget "2-Point Draw".

Detta verktyg används på fri hand.

Detta verktyg är endast tillgängligt när "komplexitet" (expertnivå) är inställd på "Avancerad".

Se även: 1-punktsritning, 1-punktsval, 2-punktsritning, 3-punktsritning, 3-punktsval, Isometrisk linje, Dimetriskt val, Dimetrisk ritning, Trimetrisk välj, Trimetrisk ritning, Sned välj, Snedritning, & Perspektiv.

Tux Paint 0.9.35
