                Rita med Tux "Magic" Verktyg: Hypotrokoid kant

                           ### Grupp Konstnärlig ###

        ## Upphovsperson: Bill Kendrick <bill@newbreedsoftware.com> ##

Klicka för att placera centrum på en hypotrochoidkurva, dra sedan åt vänster/
höger för att justera radien på en fast cirkel, och uppåt/nedåt för att justera
radien på den cirkel som kommer att rulla längs dess inre för att producera
ritningen. Den "penna" som målar kurvan kommer att befinna sig på kanten av den
rullande cirkeln.

Detta verktyg används på fri hand.

Detta verktyg erbjuder flera storleksinställningar.

Detta verktyg erbjuder färgval.

Se även: Epicykloid, Epicykloid insida, Epicykloid kant, Epicykloid utsida,
Hypotrokoid insida, Hypotrokoid insida, Hypotrokoid utsida, Stringhörn,
Strängens kanter, & String V.
