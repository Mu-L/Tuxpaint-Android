Outil 'Magie' de Tux Paint : Zoom

Groupe: Déformations de l'image

Auteur : Pere Pujal i Carabantes <pere@fornol.no-ip.org>

Cliquez et balayez vers le haut pour zoomer, et vers le bas pour dézoomer. (Quand vous dézoomer, la couleur que vous avez choisie est utilisée comme couleur d'arrière-plan)

Cet outil permet de travailler en mode à main levée.

Cet outil permet de choisir des couleurs.
