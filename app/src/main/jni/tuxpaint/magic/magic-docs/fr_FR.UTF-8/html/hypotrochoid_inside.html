Outil 'Magie' de Tux Paint : Hypotrochoïde interne

Groupe: Artistique

Auteur : Bill Kendrick <bill@newbreedsoftware.com>

Cliquez pour placer le centre d'une courbe hypotrochoïde, puis faites glisser vers la gauche/droite pour ajuster le rayon d'un cercle fixe, et vers le haut/bas pour ajuster le rayon du cercle qui roulera à l' intérieur pour produire le dessin. The "pen" that paints the curve will be on the inside of the rolling circle.

Cet outil permet de travailler en mode à main levée.

Cet outil permet de travailler avec plusieurs tailles.

Cet outil permet de choisir des couleurs.

Voir aussi : Épitrochoide, Épitrochoïde interne, Épitrochoïde de bord, Épitrochoïde externe, Hypotrochoïde, Hypotrochoïde de bord, Hypotrochoïde externe, Ficelles dans les coins, Ficelles sur les bords, & Ficelles V.
