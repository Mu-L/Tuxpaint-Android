                 Outil 'Magie' de Tux Paint : Trimetric Select

                          ### Groupe: Projections ###

           ## Auteur : Bill Kendrick <bill@newbreedsoftware.com> ##

Click (and drag) to position the pair of angles that will be used with the
"Trimetric Draw" tool.

Cet outil permet de travailler en mode à main levée.

This tool is only available with "complexity" (expertise level) set to
'Advanced'.

Voir aussi : Dessin 1 point, Choix 1 point, Dessin 2 points, Choix 2 points,
Dessin à 3-Point, Choix 3-Point, Isometric Line, Dimetric Select, Dimetric Draw
, Trimetric Draw, Oblique Select, Oblique Draw, & Perspective.
