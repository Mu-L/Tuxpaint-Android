Outil 'Magie' de Tux Paint : Hypotrochoïde

Outil 'Magie' de Tux Paint : Hypotrochoïde

Groupe: Artistique

Auteur : Bill Kendrick <bill@newbreedsoftware.com>

Cliquez pour placer le centre d'une courbe hypotrochoïde, puis faites glisser vers la gauche/droite pour ajuster le rayon d'un cercle fixe, et vers le haut/bas pour ajuster le rayon du cercle qui s'enroulera à l'intérieur pour produire le dessin. Les options de taille permettent de positionner le « crayon » qui dessine la courbe par rapport au cercle qui s'enroule. (Si la fonction 'taille' de l'outil 'Magie' est désactivée, plusieurs outils « Hypotrochoïde » seront disponibles.)

Cet outil permet de travailler en mode à main levée.

Cet outil permet de travailler avec plusieurs tailles.

Cet outil permet de choisir des couleurs.

Voir aussi : Épitrochoide, Épitrochoïde interne, Épitrochoïde de bord, Épitrochoïde externe, Hypotrochoïde interne, Hypotrochoïde de bord, Hypotrochoïde externe, Ficelles dans les coins, Ficelles sur les bords, & Ficelles V.
