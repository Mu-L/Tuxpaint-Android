              Outil 'Magie' de Tux Paint : Hypotrochoïde de bord

                          ### Groupe: Artistique ###

           ## Auteur : Bill Kendrick <bill@newbreedsoftware.com> ##

Cliquez pour placer le centre d'une courbe hypotrochoïde, puis faites glisser
vers la gauche/droite pour ajuster le rayon d'un cercle fixe, et vers le haut/
bas pour ajuster le rayon du cercle qui roulera à l' intérieur pour produire le
dessin. Le "crayon" qui dessine la courbe sera sur le bord du cercle qui
s'enroule.

Cet outil permet de travailler en mode à main levée.

Cet outil permet de travailler avec plusieurs tailles.

Cet outil permet de choisir des couleurs.

Voir aussi : Épitrochoide, Épitrochoïde interne, Épitrochoïde de bord,
Épitrochoïde externe, Hypotrochoïde, Hypotrochoïde interne, Hypotrochoïde
externe, Ficelles dans les coins, Ficelles sur les bords, & Ficelles V.
